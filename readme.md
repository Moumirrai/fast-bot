# Discord bot template

Discord bot template written in TypeScript.

## Configuration

Create `.env` file or set your env variables as follows:
```env
# Bot config
TOKEN=<YOUR_DISCORD_BOT_TOKEN>
OWNER=<OWNER_DISCORD_ID>
PREFIX=<PREFIX>
DEBUGMODE=<BOOLEAN>
RATELIMIT=<BOOLEAN>
SAVELOGS=<BOOLEAN>

# Mongodb config
MONGODB_SRV=<MONGODB_SRV>
```
